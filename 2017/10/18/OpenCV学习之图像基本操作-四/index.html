由此可知, img 为一列表,列表内嵌套列表, 此为第一层, 列表内又嵌套列表, 此为第二层, 列表内又嵌套列表, 此为第三层, 一共有3层.
第一层的每一个元素表示图片的每一行, 第二层的每一个元素表示每一行的每一列, 第三层的每一个元素表示每一个像素(由行列确定)的三通道.如下图所示:

了解了每一维的含义之后, 我们我们操控图片的像素值就容易多了
获取图片的第100行,100列处像素的三通道值 

px = img[100, 100]
print px

输出 [157 166 200]

获取100行, 100列处像素, 蓝色通道的的值: 

blue = img[100,100,0]
print blue

输出157

获得图片的属性

图片的属性包括图片矩阵的行数, 列数, 通道数, 图片的数据类型, 像素数等.

获取图片形状,输出为一元组,分别表示图片的行数,列数,通道数 

print img.shape

输出 (342, 548, 3)

获取图片大小 

print img.size

输出 562248

获取图片的数据类型,图片的数据类型很重要, 很多OpenCV-Python代码的错误就是由数据类型导致的 

print img.dtype

输出 unit8

图片区域操作

有时候,我们想要获取图片的某一特定区域, 比如眼睛, 我们通常的做法是, 先获取脸部位置, 再在脸部寻找眼睛,这样做,一方面改进了正确率, 因为眼睛必定在脸上…另一方面,提升了效率, 因为我们搜寻眼睛的范围从整个图片变成了只搜索脸部的一小块区域.
图片区域操作, 通常通过numpy的切片来完成, 下面,我们选中足球, 再把足球复制到图片的另外一个区域 

img = cv2.imread('12.jpg', -1)
ball = img[885:1040, 1035:1190]
img[885:1040, 500:655] = ball
cv2.imwrite('13.jpg', img)

分离图片通道和合并图片通道

有时候, 我们需要工作在的图片的B/G/R的某一通道, 这个时候,我们需要分离图片的BGR通道. 另外一方面, 分离了, 你就需要把他们合并在一起, 如下: 

img = cv2.imread('12.jpg', -1)
b, g, r = cv2.split(img)
cv2.imwrite('21.jpg', b)

或者,通过切片的方式,如下 

b = img[:,:,0]

或者修改图片的某一通道, 例如我们把图片的R通道置零 

img[:,:,2] = 0
