OpenCV学习之画图(三)

发表于 2017-10-18 | 阅读次数:


所有的画图函数中,会经常传入一些共同参数,总结如下:

  • img : 可以理解为一块画布,我们在这块画布上画各种东西.
  • color : 画笔的颜色, 对于 BGR 模式, 传入一个元组, 元组每一个元素代表一个通道, 对于灰度模式, 仅仅传入一个值即可,即灰度的强度.
  • thickness : 表示画笔线条的粗细. 如果我们给一个封闭图形传入这个参数的值-1, 那么这个封闭图形将会被填充,封闭图形,例如圆.
  • lineType : 线条的类型, 例如8-connected/anti-aliesed , 默认为8-connected

OpenCV学习之读取图片(一)

发表于 2017-10-17 | 阅读次数:

读一张图片

使用cv2.imread(path, flag)函数来读一张图片, 第一个参数表示图片的路径,是一个字符串,第二个参数表示读取的标识, 有以下选项可选:

  • cv2.IMREAD_COLOR : 加载一张有颜色的图片,任何透明度的设置都会被忽略,这是默认格式
  • cv2.IMREAD_GRAYSCALE : 以灰度模式加载一张图片
  • cv2.IMREAD_UNCHANGED : 不改变图片的任何格式加载图片,不会忽略透明度,也就是alpha通道
    注意: 上述flag太长,不易书写,可以用1, 0, -1来分别代替
